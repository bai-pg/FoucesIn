# 用户设置面板 - 快速开始

## 🚀 5分钟上手指南

### 1️⃣ 启动项目
```bash
cd vuepabase
npm run dev
```
访问: http://localhost:5174

### 2️⃣ 登录系统
- 使用已有账号登录,或注册新账号
- 确保登录成功(右上角显示用户信息)

### 3️⃣ 打开设置面板
在任意页面,点击右上角的 **⚙️ 设置图标**

![设置按钮位置](https://via.placeholder.com/300x60?text=顶部导航栏+☰+🌙+⚙️)

### 4️⃣ 编辑个人资料
1. 默认显示"👤 个人资料"标签页
2. 点击 **"编辑资料"** 按钮
3. 修改昵称和简介
4. 点击 **"保存"** 或 **"取消"**

### 5️⃣ 调整偏好设置
1. 切换到 **"🎨 偏好设置"** 标签页
2. 尝试切换主题(亮色/暗色)
3. 调整番茄钟时长
4. 修改会立即生效并自动保存

## 💡 核心功能速览

| 功能 | 位置 | 操作 |
|------|------|------|
| 查看资料 | 个人资料标签页 | 直接查看 |
| 编辑资料 | 个人资料标签页 | 点击"编辑资料" |
| 切换主题 | 偏好设置标签页 | 开关切换 |

## 🎯 常用场景

### 场景1: 首次使用
```
1. 登录后点击 ⚙️
2. 填写昵称和简介
3. 选择喜欢的主题
4. 设置番茄钟时长
5. 完成! ✨
```

### 场景2: 日常切换主题
```
方法1: 直接点击顶部的 🌙/☀️ 按钮
方法2: 打开设置面板 → 偏好设置 → 切换主题
```

### 场景3: 专注学习时
```
1. 打开设置面板
2. 切换到偏好设置
3. 调整专注时长(如: 45分钟)
4. 调整休息时长(如: 10分钟)
5. 关闭弹窗,开始专注!
```

## ⚠️ 注意事项

- ❗ 邮箱字段不可编辑(需通过认证流程修改)
- ❗ 所有修改需要登录状态
- ❗ 偏好设置修改后立即生效,无需刷新
- ❗ 点击遮罩层不会关闭弹窗,需点击"关闭"按钮

## 🆘 遇到问题?

### Q: 点击设置按钮没反应?
A: 检查是否已登录,未登录状态下设置按钮可能不显示

### Q: 修改后没有保存?
A: 检查浏览器控制台是否有错误信息,确认Supabase连接正常

### Q: 如何恢复默认设置?
A: 手动将各项设置改回默认值:
   - 主题: 亮色
   - 邮件通知: 开启
   - 默认视图: 仪表盘
   - 专注时长: 25分钟
   - 休息时长: 5分钟
