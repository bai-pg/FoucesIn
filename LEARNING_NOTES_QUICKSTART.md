# 学习笔记功能 - 快速开始指南

## 🚀 5 分钟快速上手

### 步骤 1: 安装依赖（已完成）
```bash
npm install @wangeditor/editor @wangeditor/editor-for-vue
```

### 步骤 2: 执行数据库迁移

1. **登录 Supabase Dashboard**
   - 访问：https://supabase.com/dashboard
   - 选择你的项目

2. **打开 SQL Editor**
   - 左侧菜单点击 "SQL Editor"
   - 点击 "New query"

3. **执行迁移脚本**
   
   复制以下内容并粘贴到编辑器中：

```sql
-- 添加 learning_notes 字段
ALTER TABLE public.study_records 
ADD COLUMN IF NOT EXISTS learning_notes TEXT;

-- 添加注释
COMMENT ON COLUMN public.study_records.learning_notes IS '富文本格式的学习笔记/反思日志，支持HTML格式';

-- 创建索引优化查询性能
CREATE INDEX IF NOT EXISTS idx_study_records_learning_notes 
  ON public.study_records(user_id) 
  WHERE learning_notes IS NOT NULL AND learning_notes != '';
```

4. **点击 "Run" 按钮执行**

5. **验证成功**
   - 看到 "Success. No rows returned" 表示成功
   - 可以运行以下查询验证：
   ```sql
   SELECT column_name, data_type 
   FROM information_schema.columns 
   WHERE table_name = 'study_records' 
     AND column_name = 'learning_notes';
   ```

### 步骤 3: 启动开发服务器
```bash
npm run dev
```

### 步骤 4: 测试功能

1. **访问学习记录页面**
   - 浏览器打开：http://localhost:3000/records

2. **找到任意学习记录**
   - 每条记录右侧有操作按钮

3. **点击 📝 图标**
   - 紫色背景的笔记图标
   - 如果已有笔记，右上角会有黄色高亮点

4. **编辑笔记**
   - 对话框标题显示科目和日期
   - 使用工具栏格式化文本
   - 输入你的学习心得

5. **保存笔记**
   - 点击底部 "💾 保存笔记" 按钮
   - 看到 "笔记保存成功!" 提示

6. **查看效果**
   - 关闭对话框
   - 笔记图标右上角出现黄色高亮标记
   - 再次点击可以编辑

## ⚠️ 常见问题

### Q1: 看不到 📝 按钮？
**A**: 
- 确保鼠标悬停在记录上（按钮默认隐藏，悬停时显示）
- 检查是否已登录
- 刷新页面重试

### Q2: 点击按钮没反应？
**A**:
- 打开浏览器控制台（F12）查看错误信息
- 确认依赖已正确安装
- 重启开发服务器

### Q3: 保存后没有高亮标记？
**A**:
- 检查数据库迁移是否成功执行
- 确认 `learning_notes` 字段已添加到表
- 刷新页面查看最新数据

### Q4: 编辑器加载很慢？
**A**:
- 首次加载需要下载编辑器资源（约 200KB）
- 后续加载会使用缓存
- 检查网络连接

### Q5: 如何删除笔记？
**A**:
- 打开笔记对话框
- 清空所有内容
- 点击保存
- 高亮标记会消失
